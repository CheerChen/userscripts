# 更新日志

## Emby 显示字段持久化修复

### v1.3.0 (2026-04-03)
- 新增跨媒体库复制字段配置功能，弹窗式 UI 面板
- 在媒体库页面的视图设置按钮旁注入复制按钮（content_copy 图标）
- 来源列表仅显示已有配置的媒体库，目标列表显示所有现存媒体库
- 支持「应用到全部」快捷操作，一键勾选所有目标媒体库
- 通过 Emby API（VirtualFolders）获取媒体库名称，兼容 `ItemId` 和 `Id` 字段
- 自动过滤已删除的媒体库（localStorage 残留条目）
- 中英双语 UI，根据浏览器语言自动切换
